# Fiducie Camille

Structure minimale pour démarrer ton site Fiducie Camille. Tu peux tout ouvrir dans Cursor et compléter design + fonctionnalités.

## Structure

```
fiducie-camille/
├── README.md
├── .gitignore
└── src/
    ├── index.html
    ├── styles/
    │   └── main.css
    ├── scripts/
    │   └── main.js
    └── assets/
        └── .gitkeep
```

## Démarrage rapide

1. Ouvre `src/index.html` dans ton navigateur ou utilise l'extension Live Server de VS Code / Cursor.
2. Mets tes images dans `src/assets/` et mets à jour les chemins dans le HTML ou le CSS.
3. Ajoute tes styles dans `src/styles/main.css` et ton interactivité dans `src/scripts/main.js`.

## Personnalisation

- Mets à jour le contenu des sections "Hero", "Services" et "Contact" dans `index.html`.
- Change les couleurs globales dans `:root` (défini dans `main.css`).
- Utilise `main.js` pour gérer des interactions (menus, formulaires, animations légères).

Bonne création !
